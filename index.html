YOLOv8-based overdimension and overload detection system for trucks

I developed an Over Dimension Detection System for trucks based on Computer Vision and Deep Learning as a solution to assist in monitoring ODOL vehicles. This system uses YOLOv8, OpenCV, and Python to detect truck loads in real-time and calculate the length and height of the load based on camera calibration. The model was trained using a dataset that was collected and labeled independently, then went through a preprocessing and augmentation process to improve accuracy. The system output displays the vehicle status (Normal or Over Dimension) along with the load dimension measurements automatically. This project integrates image processing, object detection, and digital measurement systems into one applicable solution to improve the safety of freight transportation.
